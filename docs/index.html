Model

Methodology, calibration, and WC 2022 backtest results

How it works

Predictions combine four independent models via weighted stacking:

ELO (40%) — Rating-based win probability with climate, altitude, and squad availability adjustments.

Dixon-Coles Poisson (35%) — Expected goals from attack/defense strength with low-score correlation correction.

ML Ensemble (25%) — Random Forest + Gradient Boosting trained on 110 WC/Euro matches, temperature-scaled.

H2H (8% when available) — Historical head-to-head records between specific teams.

Tournament simulation runs 100,000 Monte Carlo iterations with live ELO updates, extra-time modeling, and fatigue tracking.
